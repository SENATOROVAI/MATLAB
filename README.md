## Лабораторные работы

### Лабораторная работа №1
**Анализ динамических характеристик эталонных моделей систем со стандартной настройкой**

Исследование динамических характеристик систем автоматического управления при пяти типах стандартной настройки. Сравнительный анализ качества регулирования.

| Настройка | Перерегулирование Δy | Запас по фазе PM |
|---|---|---|
| Линейный оптимум | 16.29 % | 51.83° |
| Биномиальный оптимум | 0.00 % | 76.35° |
| Оптимум по модулю | 4.32 % | 65.53° |
| Симметричный оптимум | 54.35 % | 21.39° |
| Астатизм 3-го порядка | 56.13 % | 29.78° |

**Что сделано:**
- Получены передаточные функции замкнутых систем для каждой настройки
- Сняты кривые переходных процессов при ступенчатом воздействии
- Исследована отработка линейного `g = vt` и квадратичного `g = at²` входных сигналов
- Построены ЛАЧХ и ФЧХ, определены запасы устойчивости и показатели колебательности
- Определён порядок астатизма каждой системы

### Лабораторная работа №2
**Параметрический синтез ЦСУ с П-регулятором**

Объект управления — последовательно включённые апериодическое и интегрирующее звенья:

$$W_o(s) = \frac{1}{s(T_1 s + 1)}$$

**Что сделано:**
- Синтез методом переоборудования (ZOH), настройка на «оптимум по модулю»
- Исследование влияния периода дискретности `T₀` на качество процесса
- Определение постоянной времени `T_зап` по минимуму функционала `F = ∫|y − yэ|dt`
- Синтез с учётом вычислительной задержки `ε = T₀`
- Биномиальная настройка и сравнение с оптимумом по модулю

| Режим | T₁ | Tμ | tp1, с | Δy, % |
|---|---|---|---|---|
| Оптимум по модулю (ε = 0) | 1 | 1.4 | 10.55 | 24.13 % |
| Оптимум по модулю (ε = 0) | 0.5 | 0.9 | 8.21 | 28.80 % |
| Оптимум по модулю (ε = T₀) | 1 | 1.07 | 8.55 | 17.69 % |
| Оптимум по модулю (ε = T₀) | 0.5 | 0.57 | 4.50 | 19.10 % |
| Биномиальная (ε = 0) | 1 | 1.02 | 11.57 | 0 % |
| Биномиальная (ε = T₀) | 1 | 1.07 | 11.34 | 0 % |

### Лабораторная работа №3
**Параметрический синтез ЦСУ с И-регулятором**

Объект управления — апериодическое звено первого порядка:

$$W_o(s) = \frac{1}{T_1 s + 1}, \quad W_р(s) = \frac{K}{s}$$

**Что сделано:**
- Сравнение аналогового и цифрового И-регуляторов при постоянном и линейно нарастающем воздействиях
- Синтез цифровой системы методом переоборудования (ZOH), исследование влияния `T₀`
- Определение `T_зап` по минимуму функционала при `T_зап = 0,1T₀; 0,4T₀; 0,9T₀`
- Настройка на оптимум по модулю при `ε = 0` и `ε = T₀`
- Биномиальная настройка — апериодический процесс без перерегулирования

| T₀ | tp1, с | tp2, с | Δy, % |
|---|---|---|---|
| 0,1 Tμ1 | 2.300 | 5.900 | 18.37 % |
| Tμ1 | 1.000 | 2.000 | 39.96 % |

> Минимум функционала F достигается при **T_зап = 0,4·T₀**

## Ключевые выводы

- Увеличение периода дискретности `T₀` ухудшает качество процесса — при `T₀ = Tμ1` перерегулирование вырастает до ~40%
- Учёт вычислительной задержки `ε = T₀` в модели значительно снижает перерегулирование
- Биномиальная настройка обеспечивает апериодический процесс без перерегулирования, но увеличивает время регулирования
- Оптимум по модулю даёт лучший компромисс между быстродействием и устойчивостью
